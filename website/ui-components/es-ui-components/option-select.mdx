# Option selector

import { ComponentShowcase } from '../components/component-showcase';
import { OptionSelect, Checkbox, Button, Menu, MenuItem, MenuSeparator } from '@eightshift/ui-components';
import { icons } from '@eightshift/ui-components/icons';

This is a versatile component that can allow selecting from a set of options in various ways.

<ComponentShowcase defaultValue='option1'>
	{(data, setData) => {
		return (
			<OptionSelect
				label='Option'
				value={data}
				onChange={setData}
				options={[
					{ value: 'option1', label: 'Option 1' },
					{ value: 'option2', label: 'Option 2' },
					{ value: 'option3', label: 'Option 3' },
				]}
			/>
		);
	}}
</ComponentShowcase>

```jsx
<OptionSelect
	icon={icons.columns}
	label='Option'
	value={value} // e.g. 'option1'
	onChange={(value) => setData(value)}
	options={[
		{ value: 'option1', label: 'Option 1' },
		{ value: 'option2', label: 'Option 2' },
		{ value: 'option3', label: 'Option 3' },
	]}
/>
```

#### Submenu

Meant to be used as a part of a `Menu`.

`type='submenu'`

Most item features from `'menu'` are supported.

<ComponentShowcase defaultValue='option1'>
	{(data, setData) => {
		return (
