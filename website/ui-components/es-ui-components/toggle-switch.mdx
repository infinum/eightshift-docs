# Toggle / Switch

import { ComponentShowcase } from '../components/component-showcase';
import { Toggle, Switch, Button } from '@eightshift/ui-components';
import { icons } from '@eightshift/ui-components/icons';

<ComponentShowcase defaultValue={false}>
	{(data, setData) => {
		return (
			<Toggle
				checked={data}
				onChange={setData}
				label='Demo'
			/>
		);
	}}
</ComponentShowcase>

```jsx
<Toggle
	label='Demo'
	checked={data}
	onChange={(value) => setData(value)}
/>
```

If you need the toggle switch standalone, you can use the `Switch` component.

Other props are the same, with the exception of label and related props.

<ComponentShowcase
	defaultValue={false}
	fitWidth
>
	{(data, setData) => {
		return (
			<Switch
				checked={data}
				onChange={setData}
			/>
		);
	}}
</ComponentShowcase>

```jsx
<Switch
	checked={data}
	onChange={(value) => setData(value)}
/>
```

### Indeterminate state
Whether the toggle should be rendered in an indeterminate state.

	defaultValue={null}
	resettable
>

		return (
			<Toggle
				label='Demo'
				checked={data}
				onChange={setData}
				isIndeterminate={data === null}
			/>
		);
	}}
</ComponentShowcase>

<Toggle
