### Align icon to title

Aligns the icon vertically to the title, instead of the whole notice. Useful if the content of the notice is longer.

<ComponentShowcase
	defaultValue={false}
	preContent={(data, setData) => (
		<Checkbox
			checked={data}
			onChange={setData}
			label='Align icon to title'
		/>
	)}
>
	{(data) => (
		<Notice
			label='Lorem ipsum...'
			subtitle='dolor sit amet, consectetur adipiscing elit. Vivamus ultrices tincidunt nibh, in lacinia tellus lobortis eu. Curabitur dignissim tellus eget felis tincidunt, a maximus odio tristique.'
			type='info'
			alignIconToTitle={data}
		/>
	)}
</ComponentShowcase>

<Notice
