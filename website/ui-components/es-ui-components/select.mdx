# Single select

import { ComponentShowcase } from '../components/component-showcase';
import { Select } from '@eightshift/ui-components';
import { icons } from '@eightshift/ui-components/icons';
import { demoOptions } from './select-helpers';

<ComponentShowcase>
	{(data, setData) => (
		<Select
			value={data}
			onChange={setData}
			options={demoOptions}
		/>
	)}
</ComponentShowcase>

```jsx
<Select
	value={value}
	onChange={(value) => setValue(value)}
	options={[
		{ value: 'option-1', label: 'Option 1' },
		{ value: 'option-2', label: 'Option 2' },
		{ value: 'option-3', label: 'Option 3' },
	]}
/>
```

### Simple value

By default, values are objects with `value` and `label` properties (and optionally `metadata`).
If you want to use a simple string value, use the `simpleValue` prop.

<ComponentShowcase
	defaultValue={null}
	preContent={(data) => (
		<span className='es-uic-font-mono es-uic-text-xs esd-white-space-pre'>{JSON.stringify(data, null, 2)}</span>
	)}
>
	{(data, setData) => (
		<Select
			label='Default'
			value={data}
			onChange={setData}
			options={demoOptions}
		/>
	)}
</ComponentShowcase>
