## Misconfiguration alert

If the number of `Tab` elements and `TabPanel` elements doesn't match, an alert will be shown.

<ComponentShowcase>
	<Tabs>
		<TabList>
			<Tab>Tab 1</Tab>
			<Tab>Tab 2</Tab>
		</TabList>
		<TabPanel>Tab 1 content</TabPanel>
	</Tabs>
</ComponentShowcase>

```jsx
<Tabs>
	<TabList>
		<Tab>Tab 1</Tab>
		<Tab>Tab 2</Tab>
	</TabList>
	<TabPanel>Tab 1 content</TabPanel>
	// Missing Tab 2 content
</Tabs>
```
