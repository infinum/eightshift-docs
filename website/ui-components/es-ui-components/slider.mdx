# Slider

import { ComponentShowcase } from '../components/component-showcase';
import { Slider } from '@eightshift/ui-components';
import { icons } from '@eightshift/ui-components/icons';

<ComponentShowcase defaultValue={0}>
	{(data, setData) => {
		return (
			<Slider
				value={data}
				onChange={setData}
				aria-label='Slider'
			/>
		);
	}}
</ComponentShowcase>

```jsx
<Slider
	value={value}
	onChange={(value) => setData(value)}
/>
```

## Range slider

In order for the value display to be visible, a `label` must be set!

<ComponentShowcase defaultValue={[20, 60]}>
	{(data, setData) => {
		return (
			<Slider
				label='My slider'
				value={data}
				onChange={setData}
				aria-label='Slider'
			/>
		);
	}}
</ComponentShowcase>

```jsx
<Slider
	label='My slider'
	// highlight-next-line
	value={value} // e.g. [20, 60]
	onChange={(value) => setData(value)}
/>
```
