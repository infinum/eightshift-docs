# Customization

import { ComponentShowcase } from '../components/component-showcase';
import { Select, AsyncSelect, MultiSelect, AsyncMultiSelect, HStack } from '@eightshift/ui-components';
import { icons } from '@eightshift/ui-components/icons';
import { demoOptions, demoGetData } from './select-helpers';

Select components allow many customizations to adapt to various use cases.

For the complete list of customizations, use your IDE's autocomplete functionality. Look for props starting with `custom`.

Most common customizations are listed below.

## Custom dropdown arrow

_Applies to all select components_

<ComponentShowcase>
	{(data, setData) => (
		<Select
			value={data}
			onChange={setData}
			options={demoOptions}
			customDropdownArrow={icons.arrowDownCircle}
		/>
	)}
</ComponentShowcase>

```jsx
<Select
	value={value}
	onChange={(value) => setValue(value)}
	options={[
		{ value: 'option-1', label: 'Option 1' },
		{ value: 'option-2', label: 'Option 2' },
		{ value: 'option-3', label: 'Option 3' },
	]}
	// highlight-next-line
	customDropdownArrow={icons.arrowDownCircle}
/>
```

## Custom menu option

_Applies to all select components_

<ComponentShowcase defaultValue='item-1'>
	{(data, setData) => (
		<Select
			value={data}
			onChange={setData}
			options={demoOptions}
			customMenuOption={(item) => <span className='es:font-bold es:text-blue-400 es:text-sm'>{item?.label}</span>}
			simpleValue
		/>
	)}
</ComponentShowcase>

```jsx
<Select
	value={value}
	onChange={(value) => setValue(value)}
	options={[
		{ value: 'option-1', label: 'Option 1' },
		{ value: 'option-2', label: 'Option 2' },
		{ value: 'option-3', label: 'Option 3' },
	]}
	// highlight-next-line
	customMenuOption={(item) => <span className='es:font-bold es:text-blue-400'>{item?.label}</span>}
/>
```

## Custom value display (multi-select)

_Applies to `MultiSelect` and `AsyncMultiSelect`_

Only applies to cases where 1 item is selected.

<ComponentShowcase defaultValue={[demoOptions[0]]}>
	{(data, setData) => (
		<MultiSelect
			value={data}
			onChange={setData}
			options={demoOptions}
			customValueDisplay={(item) => (
				<HStack
					className='es:icon:size-[1em] es:font-bold es:text-blue-400'
					slot='label'
				>
					{icons?.[item?.icon] ?? item?.icon}
					{item?.label}
				</HStack>
			)}
		/>
	)}
</ComponentShowcase>

```jsx
<MultiSelect
	value={value}
	onChange={(value) => setValue(value)}
	options={[
		{ value: 'option-1', label: 'Option 1' },
		{ value: 'option-2', label: 'Option 2' },
		{ value: 'option-3', label: 'Option 3' },
	]}
	// highlight-start
	customValueDisplay={(item) => (
		<HStack
			className='es:icon:size-[1em] es:font-bold es:text-blue-400'
			slot='label'
		>
			{item?.icon}
			{item?.label}
		</HStack>
	)}
	// highlight-end
/>
```
