# Placeholders

import { ComponentShowcase } from '../components/component-showcase';
import {
	ImagePlaceholder,
	FilePlaceholder,
	MediaPlaceholder,
	Checkbox,
	OptionSelect,
	RichLabel,
} from '@eightshift/ui-components';
import { icons } from '@eightshift/ui-components/icons';

For the complete list of props, use your IDE's autocomplete functionality.

## Image placeholder

<ComponentShowcase
	defaultValue={false}
	preContent={(data, setData) => (
		<Checkbox
			label='Show image'
			checked={data}
			onChange={setData}
		/>
	)}
>
	{(data) => <ImagePlaceholder url={data === true ? 'https://picsum.photos/200' : null} />}
</ComponentShowcase>

```jsx
<ImagePlaceholder url='https://picsum.photos/200' />
```

### Image display mode

<ComponentShowcase
	defaultValue='cover'
	preContent={(data, setData) => (
		<OptionSelect
			label='Image display mode'
			value={data}
			onChange={setData}
			options={[
				{ value: 'contain', label: 'Contain', subtitle: <span className='es:font-mono'>contain</span> },
				{ value: 'cover', label: 'Cover', subtitle: <span className='es:font-mono'>cover</span> },
				{ value: 'fill', label: 'Fill', subtitle: <span className='es:font-mono'>fill</span> },
			]}
			type='menu'
		/>
	)}
>
	{(data) => (
		<ImagePlaceholder
			url='https://picsum.photos/900/380'
			imageMode={data}
		/>
	)}
</ComponentShowcase>

```jsx
<ImagePlaceholder
	url='https://picsum.photos/900/380'
	// highlight-next-line
	imageMode='contain'
/>
```

### No image (placeholder only)

This can also be achieved by not providing an `url`, but this should be
a bit more performant as you don't have an `<img>` without a source.

<ComponentShowcase>
	<ImagePlaceholder noImage />
</ComponentShowcase>

```jsx
<ImagePlaceholder
	// highlight-next-line
	noImage
/>
```

## Media placeholder

`style` and `size` works the same as in the `ImagePlaceholder`, so they won't be covered again.
