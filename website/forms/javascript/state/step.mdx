The `step` object contains methods used in multi-step/multi-flow forms.

:::tip
For more details about this feature visit [multi-flow](/forms/features/multiflow-forms) or [multi-step](/forms/features/multistep-forms) features.
:::

<WindowObject name="Step" filter="step" />

### Example

This example will change the form step to the provided step `id`.

```js
window?.esForms?.step?.goToNextStep('<formId>', '<nextStepId>');
```
