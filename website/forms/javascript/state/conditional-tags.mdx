The `conditionalTags` object contains methods used to manupulate conditional logic.

:::tip
For more details about this feature visit [conditional-tags](/forms/features/conditional-tags) feature.
:::

<WindowObject name="Conditional tags" filter="conditionalTags" />

### Example

This example will set conditional logic to field.

```js
window?.esForms?.conditionalTags?.setField('<name>', '<formId>');
```
