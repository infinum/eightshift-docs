### ES_API_KEY_MOMENTS

This variable will set the Moments integration API key.

```php
define('ES_API_KEY_MOMENTS', '<api-key>');
```

### ES_API_URL_MOMENTS

This variable will set the Moments integration API URL.

```php
define('ES_API_URL_MOMENTS', '<api-url-key>');
```
