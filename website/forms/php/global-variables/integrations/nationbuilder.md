### ES_CLIENT_SLUG_NATIONBUILDER

This variable will set the NationBuilder client slug (your NationBuilder nation slug).

```php
define('ES_CLIENT_SLUG_NATIONBUILDER', '<client-slug>');
```

### ES_CLIENT_ID_NATIONBUILDER

This variable will set the NationBuilder OAuth client ID.

```php
define('ES_CLIENT_ID_NATIONBUILDER', '<client-id>');
```

### ES_CLIENT_SECRET_NATIONBUILDER

This variable will set the NationBuilder OAuth client secret.

```php
define('ES_CLIENT_SECRET_NATIONBUILDER', '<client-secret>');
```
