### ES_API_KEY_CORVUS_&#123;storeId&#125;

This variable will set the Corvus integration API key for a specific store. Replace `STORE_ID` with the store ID in uppercase (e.g. `ES_API_KEY_CORVUS_123`).

```php
define('ES_API_KEY_CORVUS_<store-id>', '<api-key>');
```
