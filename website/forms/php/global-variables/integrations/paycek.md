### ES_API_KEY_PAYCEK

This variable will set the Paycek integration API key.

```php
define('ES_API_KEY_PAYCEK', '<api-key>');
```

### ES_PROFILE_KEY_PAYCEK

This variable will set the Paycek integration profile key.

```php
define('ES_PROFILE_KEY_PAYCEK', '<api-key>');
```
