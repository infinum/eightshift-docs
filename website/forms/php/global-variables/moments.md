### ES_API_KEY_MOMENTS

This variable will set Moments integration api key.

```php
define('ES_API_KEY_MOMENTS', '<api-key>');
```

### ES_API_URL_MOMENTS

This variable will set Moments integration api url key.

```php
define('ES_API_URL_MOMENTS', '<api-url-key>');
```
